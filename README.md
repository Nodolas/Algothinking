# K-Means Clustering — Algorithm Deep Dive

An interactive presentation on the K-Means clustering algorithm, built as a self-contained static site.

## Hosting on GitHub Pages

1. Push this repository to GitHub (if not already done):
   ```bash
   git add index.html README.md
   git commit -m "Add GitHub Pages presentation"
   git push
   ```

2. In your GitHub repository, go to **Settings → Pages**.

3. Under **Source**, select **Deploy from a branch**, choose the `main` branch and `/ (root)` folder, then click **Save**.

4. After ~1 minute your site will be live at:
   ```
   https://<your-username>.github.io/<your-repo-name>/
   ```

## Fullscreen Presentation Mode

- Click the **Fullscreen** button in the top-right of the nav bar, or
- Press **`F`** anywhere on the page to toggle fullscreen.
- Press **`Esc`** to exit fullscreen.

Scroll through sections using the mouse wheel or arrow keys while in fullscreen for a clean, distraction-free presentation.
